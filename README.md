# NEONEXUS-MentalHealth-platform
NEONEXUS is an AI-powered mental health platform designed to support students in higher education in India. It provides accessible, private, and culturally relevant mental health assistance.

Overview:
- AI chatbot for mental health support
- Multilingual support (English, Hindi, Urdu, Kashmiri, Dogri)
- Standard mental health assessments (PHQ-9, GAD-7, GHQ-12)
- Crisis detection and support features
- Privacy-focused (anonymous and secure)

Key Features:
- Student-focused platform
- AI-driven chatbot using NLP models
- Easy-to-use web interface
- Confidential and secure interactions
